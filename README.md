# CODSOFT Java Development Internship

This repository contains the tasks completed during my **Java Development Virtual Internship at CodSoft**.  
The internship focuses on building basic Java applications to strengthen programming logic, problem-solving skills, and understanding of core Java concepts.

During this internship, I developed multiple console-based applications using Java and practiced concepts such as loops, conditional statements, object-oriented programming, and user input handling.

---

## Task 1 – Number Game

The Number Game is a simple Java program where the computer generates a random number within a specified range (1–100). The user attempts to guess the number by entering guesses through the console.

The program provides feedback indicating whether the guess is too high or too low, helping the user adjust their next guess. The game continues until the correct number is guessed.

### Features
- Random number generation
- User input handling
- Conditional feedback (Too High / Too Low)
- Loop execution until the correct answer is found

### Concepts Used
- Java Random class
- While loops
- Conditional statements
- Scanner class for input

## Task 2 – Student Grade Calculator

The Student Grade Calculator is a Java application that calculates the total marks and average percentage of a student based on marks entered for multiple subjects.

The program then assigns a grade based on the calculated average percentage using conditional statements.

### Features
- Accepts marks for multiple subjects
- Calculates total marks
- Computes average percentage
- Assigns grades based on performance

### Grade Criteria Example
- 90 and above – Grade A  
- 75–89 – Grade B  
- 60–74 – Grade C  
- 50–59 – Grade D  
- Below 50 – Grade F  

### Concepts Used
- For loops
- Arithmetic operations
- Conditional statements
- User input using Scanner

## Task 3 – ATM Interface

The ATM Interface simulates basic ATM functionalities using Java classes and methods. The program allows the user to interact with a simple ATM system through a menu-driven interface.

Users can check their balance, deposit money, or withdraw funds.

### Features
- Balance checking
- Deposit money
- Withdraw money
- Menu-driven console interface

### Concepts Used
- Object-Oriented Programming (OOP)
- Classes and objects
- Methods
- Conditional statements
- User input handling
